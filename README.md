# Wordcount
This is a sample App for the [Mister](https://github.com/davidfregoli/mister) MapReduce framework

### Sample input:
- file `input-one`:
  ```
  this is the first file
  ```
- file `input-two`:
  ```
  this is certainly the second file
  ```
### Sample output:
```
this 2
is 2
the 2
first 1
file 2
certainly 1
second 1
```
